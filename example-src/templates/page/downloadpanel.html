Download Theme

Once your happy with the changes, click the Download Buttons below and you will get the Theme customisations (in JSON or CSS)

The theme.css file will be a modified bootstrap.css file, so you can replace your bootstrap.css file (if you have one) with the theme.css file and your customised theme will be used.

The theme.json file can be used to import the customisations back into Cluckles at a later point, so you dont have to start customising from scratch again!

I also developed the BoostrapThemeBundle for Symfony2, which integrates Bootstrap Themeing for Symfony apps (which embeds Cluckles).
