BootstrapThemeEditor

Bootstrap Theme Editor allows live modification of Bootstrap themes so that you can customise them easily. There is an example page provided (build/example/index.html) which makes it easy to see it in action!

The example deme page has a User Interface which has panels where you can change the different parts of bootstrap by using color pickers to change the styling. Simple and Easy.

The included files can be distributed and used in other projects to provide live editing functionality for Bootstrap.

I made this tool because I needed a way to Live edit Bootstrap in Corvus. I couldnt find anything existing which I could integrate into the project, so I made BootstrapThemeEditor.

If you find the project usefull, feel free to fork it and extend it, or even dontate to the project with Gittip
