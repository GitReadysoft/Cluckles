Bootstrap Theme Editor

Bootstrap Theme Editor allows live modification of Twitter Bootstrap themes so that you can customise them easily.

The included files can be distributed and used in other projects to provide Live editing functionality for Bootstrap.

I made this tool because I needed a way to Live edit Bootstrap in Corvus. I couldn't find anything existing which I could integrate into the project, so I made Bootstrap Theme Editor.

If you find the project useful, feel free to fork it and extend it, or even donate to the project with Gittip
