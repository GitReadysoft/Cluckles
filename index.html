Cluckles

Cluckles is a live theme editor that leverages Less to Customise Twitter Bootstrap (In the future I will look into adding support other CSS Frameworks).

New! You can directy import a variables.less file and Cluckles will take care of the rest!
You can also add Custom Less/Css to further customise your theme!

The included JS files can be distributed and used in other projects to provide Live editing functionality for Bootstrap. (An example site is also provided). P.S. This website was completely Themed using Cluckles!

Getting started with Cluckles

To get started using Cluckles download the latest distribution file or browse the Releases Now load up the Example site which is included or start composing your own: 

 
            <!DOCTYPE html>
            <html>
              <title>Example Cluckles</title>

              // Include the Customised Bootstrap.less file, which includes custom variables
              // build/less/bootstrap.less
              <link type="text/css" href="../less/bootstrap.less" rel="stylesheet/less" />
            </head>
            <body>
              // Page Markup and such

              // Include Bootstrap and jQuery which are needed
              <script src="//ajax.googleapis.com/ajax/libs/jquery/1.11.1/jquery.min.js"></script>
              <script src="//maxcdn.bootstrapcdn.com/bootstrap/3.1.1/js/bootstrap.min.js"></script>

              // Configure the Less Global Object (Useful for Debugging)
              <script>
                less = {
                    env: "development",
                    logLevel: 2,
                    async: false,
                    fileAsync: false,
                    poll: 1000,
                    functions: {},
                    dumpLineNumbers: "comments",
                    relativeUrls: false,
                };
              </script>

              // Now load Less so we can process it
              <script src="//cdnjs.cloudflare.com/ajax/libs/less.js/1.7.0/less.min.js"></script>
              
              // Load our Fancy live editing Script
              <script src="../cluckles-0.10.0.js"></script>
              <script>
                var clucklesEditor = new ClucklesEditor(less, {
                    export: {
                        scope: {
                          selector: "#mainContainer",
                          customCss: true,
                          customLess: true
                        },
                        target: '#download-panel-footer', // Fallback/General Target
                        json: {
                            target: '#download-panel-footer', // Custom Target
                            id: 'download-theme-json', // Custom Id
                            text: 'Download Json' // Custom Text
                        },
                        css: {}, // Blank Object provided makes the Css download button appear
                        save: {
                            url: "http://localhost:9000/example/",
                            callback: function() {
                                alert('Theme modifications have been saved');
                            }
                        }
                    }
                });
              </script>
            </body>

Now Cluckles is loaded!

So we can go ahead and start using it. Cluckles has several API Components which can be used to interact with Bootstrap and change the styling. An example would be changing the body background color.

            // Through a page element, using the data-cluckles-{type}={bind-property} two way databinding
            <input type="color" data-cluckles-misc="body-bg" />
        

            <script>
              // Or Programmatically (type it in your browser's console/script)
              clucklesEditor.misc.setBackgroundColor('#BFFF85');

              // Or manipulate the property manually
              clucklesEditor.misc.bodyBg.value = '#BFFF85';
              clucklesEditor.applyModifications();
            </script>

Now be amazed as Cluckles restyles the page, and from this point you can get your Changes. If you have provided export options when you called new ClucklesEditor(options);, you will have download buttons, if not you can programmatically extract the changes:

          <script>
            // Will give you the JSON of the modified parts
            // { "@bodyBg": "#BFFF85"}
            clucklesEditor.getJSON();
            
            // Property which holds the Compiled Css
            clucklesEditor.export.compiledCss;
          </script>
